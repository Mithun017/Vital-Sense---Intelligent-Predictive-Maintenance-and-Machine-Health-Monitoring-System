# Vital Sense AI - Future Dashboard Roadmap

This document outlines the strategic expansion of the Vital Sense visualization layer. All features are intended to be modular additions that preserve the integrity of the core predictive maintenance engine.

## Phase 1: Main Overview Panel
- **Machine Status**: Normal/Warning/Critical
- **Health Score**: 0-100% calculation
- **Active Alerts**: Counter for pending issues

## Phase 2: Real-time Sensor Visualization
- **Live Graphs**: Temperature, Vibration, Current, Voltage
- **No-reload Sync**: Smooth WebSocket or polling updates

## Phase 3: Failure Prediction Panel
- **Risk Level**: Low/Medium/High classification
- **Trend Indicator**: Directional health movement

## Phase 4: Remaining Useful Life (RUL)
- **Time Estimation**: Hours/Days countdown
- **Visual Progress**: Gauge/Bar representation

## Phase 5: AI Explanation Panel
- **XAI Reasoning**: Feature importance (Temp/Vibe/Current)
- **Natural Language**: Human-readable diagnostic "Why"

## Phase 6: Maintenance Recommendation
- **Actionable Directives**: Suggested interventions
- **Prioritization**: High-impact tasks first

## Phase 10: Energy & Efficiency Panel (NEW)
- **Power Consumption**: Real-time wattage (V * I)
- **Efficiency Index**: Percentage-based performance score
- **Optimization**: AI suggestions for energy saving

## Phase 11: Simulation Mode
- **Fault Injection**: Manual control for testing system response

## Phase 12: AI Chat Assistant
- **Interactive Queries**: Voice/Text interface for machine history

---
*Note: This roadmap is used for incremental expansion of the Vital Sense platform.*
